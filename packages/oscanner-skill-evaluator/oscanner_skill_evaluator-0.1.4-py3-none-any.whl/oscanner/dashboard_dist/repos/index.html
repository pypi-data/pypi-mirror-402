Multi-Repository Analysis

Extract commit data and find common contributors across 2-5 GitHub/Gitee repositories

Instructions
  • Enter between 2 and 5 GitHub/Gitee repository URLs
  • Each URL should be on a separate line
  • Supports HTTPS format: https://github.com/owner/repo or https://gitee.com/owner/repo
  • Click Analyze to extract commit data and find common contributors
  • Existing repositories will be skipped automatically
