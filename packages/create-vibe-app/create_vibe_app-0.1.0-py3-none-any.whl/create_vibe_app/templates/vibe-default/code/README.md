## Structure
Organize your code based on your project type:

### Backend
```
code/
├── src/
├── tests/
└── config/
```

### Frontend
```
code/
├── src/
│   ├── components/
│   ├── pages/
│   └── utils/
└── public/
```

### Full Stack
```
code/
├── backend/
├── frontend/
└── shared/
```

## Tips
- Follow consistent naming conventions
- Keep related code together
- Write tests alongside implementation
