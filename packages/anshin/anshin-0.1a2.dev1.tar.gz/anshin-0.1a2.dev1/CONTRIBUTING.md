# Contributing

>>> [!important]
By contributing, you certify the DCO *AND* agree your contributions are
licensed under __Apache-2.0 AND GPL-v3-or-later__.
>>>

## Sign Your Commits!

__All commits must be signed.__

By signing your commit, you agree to the terms set forth in the `DCO-1.1.txt`
file available at this project's root *OR* hosted at
https://developercertificate.org/ (Version 1.1 ONLY)

Per the Linux Foundation[<sup>1</sup>][dco]:

> When creating a commit with the Git CLI, a sign-off can be added with the -s
> option: https://git-scm.com/docs/git-commit#git-commit--s. The sign-off is
> stored as part of the commit message itself, as a line of the format:
>
> `Signed-off-by: Full Name <email>`

More information about the Developer Certificate of Origin (DCO) is available
[here][dco]
