# LI-COR LI-190R PAR Sensor

## Overview

The LI-190R is a quantum sensor that measures Photosynthetically Active Radiation (PAR) in the 400-700 nm wavelength range, which is critical for understanding photosynthetic processes in marine environments.

## Key Features

- **Manufacturer**: LI-COR Biosciences
- **Model**: LI-190R
- **Type**: Radiation Sensor (PAR)
- **Deployment**: Wing-mounted, upper side

## Measured Variables

- Photosynthetically Active Radiation (PAR_AIR)

## Specifications

| Parameter | Value |
|-----------|-------|
| Range | 0-10,000 µmol s⁻¹ m⁻² |
| Spectral Range | 400-700 nm |
| Calibration Uncertainty | ±5% |
| Response Time | <1 ms |

## Deployment Notes

Mounted on the upper side of the Saildrone wing to measure incoming photosynthetically active radiation. The sensor's fast response time captures rapid changes in light conditions.
