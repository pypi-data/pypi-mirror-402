# Apogee SI-111 Infrared Radiometer

## Overview

The SI-111 is a narrow field-of-view infrared radiometer designed to measure sea surface temperature remotely, providing non-contact temperature measurements of the ocean surface.

## Key Features

- **Manufacturer**: Apogee Instruments
- **Model**: SI-111
- **Type**: Radiation Sensor (Infrared)
- **Deployment**: Wing-mounted, lower side, viewing sea surface

## Measured Variables

- Sea surface infrared temperature (TEMP_IR_SEA_WING_UNCOMP)

## Specifications

| Parameter | Value |
|-----------|-------|
| Temperature Range | -40 to +80°C |
| Accuracy | ±0.2°C (at 15-35°C) |
| Field of View | 22° half angle |
| Spectral Range | 8-14 µm |

## Deployment Notes

Mounted on the lower side of the Saildrone wing, pointing downward at the sea surface. The narrow field of view and infrared measurement provide non-contact sea surface temperature data complementary to the hull-mounted CTD.
